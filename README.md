## Шаблон для быстрого старта телеграм бота
Используемые версии:
```
- Python 3.14
- python-telegram-bot 22.7
- dotenv 0.9.9
```

Для запуска нужно создать `.env` файл с параметрами:
```
TELEGRAM_BOT_TOKEN=<TOKEN>
CALLBACK_DATA_SEPARATORS=_:; # Здесь можно подряд указывать разделители, используемые в `callback_data`, либо не указывать и переписать логику `query_handler` в 97-ой строке. Нужны чтобы отсеять статичные клавиатуры или быстрые запросы.
```
